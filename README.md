# spring-cloud-ecosystem
클라우드 서비스에서 제공하는 분산 컴퓨팅을 활용하기 위한 개념인 클라우드 네이티브에 대해 학습하고, 이를 Spring 생태계 내에서 쉽게 활용할 수 있도록 Spring Cloud 생태계에 대한 학습 자료입니다.
